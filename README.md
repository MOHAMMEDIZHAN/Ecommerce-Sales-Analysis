# 📊 E-Commerce Sales Insights Analysis

An end-to-end data analytics project that analyzes e-commerce sales data using **MySQL** and visualizes business insights through an **interactive Tableau dashboard**. The project focuses on uncovering sales trends, customer behavior, product performance, regional insights, and revenue growth to support data-driven business decisions.

---

## 📌 Project Overview

This project demonstrates how SQL and Tableau can be combined to transform raw transactional data into meaningful business insights.

The workflow includes:

- Data exploration 
- Business-driven SQL analysis
- KPI generation
- Interactive dashboard creation
- Sales and profitability analysis

---

## 🛠️ Tech Stack

- **Database:** MySQL
- **Visualization:** Tableau
- **Dataset:** E-Commerce Sales Dataset
- **Concepts Used:**
  - Aggregate Functions

## 📌 Key Insights

- Home Decor generated the highest sales among all product categories.
- Revenue is distributed relatively evenly across categories, indicating a diversified product portfolio.
- Net Banking and Cash on Delivery were among the most preferred payment methods.
- The highest-selling product generated over **$850K** in sales.
- Top 20% products contributed approximately 80% of sales.
- North region generated the highest profit.
- Bangalore recorded the highest city-wise sales, followed closely by Lucknow and Guwahati.
- Sales increased significantly from **2023 to 2024** before declining in **2025**, highlighting changing business trends.
- Average discount remained around **10%**, helping maintain profitability while supporting sales.
